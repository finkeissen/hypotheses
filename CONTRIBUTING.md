# Contributing

## Neue Hypothese hinzufügen
1. Nächste freie ID wählen: `H###`
2. Ordner anlegen: `H###/`
3. Dateien aus `H001/` kopieren und ausfüllen.
4. `index.md` und ggf. `HYPOTHESES.md` aktualisieren.
5. Keine Modelle, kein Code, keine Runs: das gehört nach `predictions`/`predictions`.

## Status
- `draft`: formuliert, aber noch nicht im aktiven Gebrauch
- `active`: wird für Instanziierungen/Projektionen verwendet
- `falsified`: widerlegt (nicht löschen)
- `archived`: obsolet/ersetzt (nicht löschen)
